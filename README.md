## PizzaHut-Sales-SQL-Analysis
### Project Overview: 
This project analyzes Pizza Hut sales data using SQL to extract meaningful business insights. It focuses on understanding sales performance, customer behavior, and product trends.
### Database Description:  
The dataset contains the following tables:
- Orders – Order ID, date, and time
- Order Details – Quantity and pizza IDs
- Pizzas – Price and size
- Pizza Types – Name and category
### Objectives:  
The project answers key business questions such as:  
- Total number of orders
- Total revenue generated
- Highest priced pizza
- Most popular pizza size
- Top 5 most ordered pizzas
- Category-wise sales distribution
- Orders by time (hour-wise analysis)
- Average pizzas ordered per day
- Top pizzas by revenue
- Revenue contribution by category
- Cumulative revenue over time
- Top pizzas per category
### SQL Concepts Used:  
- SELECT, JOIN
- GROUP BY, ORDER BY
- Aggregate Functions (SUM, COUNT, AVG)
- Window Functions (RANK, OVER)
- Subqueries
### Key Insights:  
- Identified top-selling pizza categories
- Analyzed peak order hours
- Measured revenue contribution of each category
- Tracked sales growth over time
### Conclusion:  
This project demonstrates how SQL can be used to analyze business data and generate actionable insights for decision-making.
